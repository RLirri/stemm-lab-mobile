# STEMM Lab  
## Context-Aware Mobile Learning Platform for Experiential STEM Education

STEMM Lab is a context-aware mobile application designed to support experiential learning through hands-on STEMM (Science, Technology, Engineering, Mathematics, and Medicine) activities. The application enables students to complete real-world challenges by leveraging mobile device capabilities such as sensors, GPS, media capture, background processing, and cloud-based data synchronisation.

This project is developed as part of a mobile application development subject and follows professional software engineering practices, including Agile development, layered architecture, secure configuration management, and comprehensive testing.

## Project Objectives

The primary objectives of the STEMM Lab application are to:

- Enable students to complete structured STEMM activities using real-world data
- Demonstrate the use of context-aware computing (sensors, location, background tasks)
- Apply professional mobile application architecture and development practices
- Evaluate usability and accessibility in mobile interface design
- Deliver a production-ready mobile application with testing and deployment evidence


## Key Features

- Multi-screen mobile application with clear navigation
- Context-aware features using device sensors and GPS
- Media capture (image/video) with structured activity submission
- Offline-first data storage using SQLite
- Cloud synchronisation and authentication using Firebase
- Background tasks and notifications for improved reliability
- Battery-aware and performance-conscious design
- Comprehensive testing (unit, integration, E2E, Firebase Test Lab)
- Release-ready Android APK build


## Technology Stack

- **Framework:** React Native (JavaScript / TypeScript)
- **Navigation:** React Navigation
- **Local Storage:** SQLite
- **Cloud Backend:** Firebase (Authentication, Firestore, Test Lab)
- **Sensors & Location:** Device accelerometer, gyroscope, GPS
- **Background Processing:** WorkManager / background task libraries
- **Notifications:** Local notifications
- **Advertising:** Google AdMob
- **Testing:** Jest, End-to-End testing, Firebase Test Lab
- **Version Control:** Git + GitHub
- **Project Management:** Jira (Scrum methodology)

## Development Methodology

This project follows an Agile Scrum-based approach with multiple development sprints aligned to assessment milestones. All work is tracked using Jira, including epics, user stories, tasks, testing, and evidence capture.

Each sprint includes:
- Clearly defined sprint goals
- Traceable user stories linked to assessments
- Incremental feature delivery
- Regular commits linked to Jira issue IDs


## Build and Deployment

The application is built as an Android APK suitable for installation on physical devices. Build instructions and deployment notes are documented as part of the testing and deployment report.
